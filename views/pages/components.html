Expressful

Server & Front-end Framework from the Future.

Getting Started

Expressful sets up a professional website for you. You just write HTML and CSS.

1. Install Node.

2. Install the project generator through the terminal. sudo might be neccesary. 

$ npm install -g create-expressful-app

3. Run the create-expressful-app command followed by your project name. 

$ create-expressful-app my-website

4. Go to your newly created project directory and run the server. 

$ cd my-website
$ npm run dev

5. Learn more about Expressful.
